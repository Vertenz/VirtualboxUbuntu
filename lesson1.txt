Создание виртуальной ОС:
1. Скачать установочный фаил Ubuntu
2. Скачать и установить VirtualBox
3. Используя VirtualBox уставновить Ubuntu (при установке указать путь к ранее загруженному файлу)
5. Запусть виртуальную машину.

Подготовка убунту для работы:
1. Установить необходимые пакеты 
	a) запустить консоль (cntr + alt + t)
	б) ввести команду ' sudo apt-get update '  (для обновления индекса пакетов) `
	в) ввести команду ' sudo apt-get install gcc make perl '  (для установки необходимых пакетов) (на появившийся вопрос ответить ' y ') `

2. Установить VBox_GAs
	а) в Oracle VM VirtualBox нажать ' Устройства ' => ' Подключить образ диска Дополнений гостевой ОС '
	в) непосредсвенно в ubuntu запустить с рабочего стола ' VBox_GAs_6.1.4 '
	г) перезагрузить виртуальную ОС, для выключения OC в консоле пишем sudo poweroff (для перезагрузки sudo reboot) `

3. Для использования общего буффера обмена открыть Oracle VB нажать ' Настройки ' => ' Дополнительно ' => ' Общий буфер и обмена  ' и ' Функция DragnDrop ' указать ' Двунаправленный '

4. Установить текстовый редактор Vim. В консоле ввести команду ' sudo apt-get install Vim '. (можно использовать ключ ' -y ', чтобы сразу ответить положительно на все возникающие в консоле вопросы) `

5. Установка SSH:
	a) в вертуально ОС в консоле прописать ' sudo apt-get install openssh-server ' `
	б) если windows, то открываем Oracle VM ' Настроить ' => ' Сеть ' => ' Адаптер 2 ' => ' Тип подключения  ' установить ' Виртуальный адаптер хоста '
	в) запустить консоль в вашей ОС (для windows PowerShell ). Ввести ' ssh имя@ip ' `

6. Установить SSH ключ
 	а) в Ubuntu в терминале ввеси ' ssh-keygen '
	б) получившиеся ключи скопировать в вашу ОС
	в) создать фаил authorized_keys c данными из id_rsa.pub. В консоле пишем ' cp путь/id_rsa.pub  путь/authorized_keys ' `
	г) при подключении прописать ' ssh имя@ip -i ~/путь к ключу ' `


7.(дополнительно) Отключение ввода пароля при sudo. Открыть ' sudo visudo ' и добавить сроку ' имя ALL=(ALL:ALL)  NOPASSWD:ALL ' `

8. Добавить в исключение порт 22. 1. Для включения UFW ввести в терминале ' sudo ufw enable ' . Чтобы разрешить порт ' sudo ufw allow 22 ' `

9. Если не пускает пользователя root по SSH:
	a) создать новый пароль для пользователя команда ' sudo passwd root '

	б) открыть настройки SSH 'vi путь/sshd_config ' изменяем строку ' PermitRootLogin ' - значение ' yes '

10. Изменения motd и sshd_config
